### serverManager
A server information management tool developed by python Tkinter

## serverManager是什么？
一款主机运维管理工具，主机资产管理平台，支持导入导出，批量添加。

## 开发语言与框架：
 * 编程语言：Python3.8.5
 * 前端：tkinter
 * 数据库：sqlite3

 ## serverManager环境要求：
 * 编程语言：Python3.8.5
 * 操作系统：Windows 10 64位专业版 ltsc
